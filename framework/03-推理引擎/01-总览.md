# 推理引擎总览

> 包含逻辑层、贝叶斯层、信息论层、控制论层、图灵机层五层

## 架构

```
推理引擎
├── 逻辑层   — 蕴含图 + derives_from链 + 形式推导
├── 贝叶斯层 — 信念传播 + 置信度连续化
├── 信息论层 — KQI指数 + 香农熵
├── 控制论层 — PID反馈 + 自适应阈值
└── 图灵机层 — 知识状态机 + 快照
```

## 与MOF四层的映射

| MOF层 | 引擎组件 | 说明 |
|-------|---------|------|
| M3 元元模型层 | 形式语言层 | BNF定义推理规则的语法 |
| M2 元模型层 | 逻辑层 | 推导关系的类型约束 |
| M1 模型层 | 贝叶斯+信息论层 | 具体推导实例的置信度和度量 |
| M0 实例层 | 控制论+图灵机层 | 执行和反馈 |
