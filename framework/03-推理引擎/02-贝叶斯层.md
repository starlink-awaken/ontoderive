# 贝叶斯层——信念传播与置信度管理

> 理论来源：贝叶斯统计 (Bayes, 1763) | 实现位置: engine/bayesian.py
> 集成规约: C-09 (engine/derive.py)

---

## 一、设计目标

将置信度从离散标签(fact/high/medium/low)升级为连续概率 P(e) ∈ [0,1]。
当事实基座更新时，信念自动沿 derives_from 链传播。

## 二、置信度映射

| 离散标签 | 连续概率 | 含义 |
|---------|---------|------|
| fact | 0.95 | 多次验证的事实 |
| high | 0.92 | 高置信度推论 |
| inference | 0.85 | 正常推论 |
| medium | 0.70 | 中等置信度 |
| hypothesis | 0.50 | 假设 |
| low | 0.30 | 低置信度 |
| estimated | 0.25 | 估算值 |
| assumption | 0.10 | 假定 |

## 三、信念传播算法

### 直接推导传播
```
给定: INF derives_from [D-F1, D-F2, D-F3]
计算: P(INF) = avg(P(D-F1), P(D-F2), P(D-F3)) × 0.9
```

### 间接推导传播(INF→INF)
```
给定: INF2 derives_from [INF1]
计算: P(INF2) = P(INF1) × 0.8
迭代直到收敛或达最大迭代次数(10)
```
