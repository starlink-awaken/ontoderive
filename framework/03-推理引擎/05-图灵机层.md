# 图灵机层——知识状态机(K-TM)

> 理论来源：图灵机模型 (Turing, 1936) | 实现位置: engine/turing_k.py
> 集成规约: C-12

## 形式化定义

```
K-TM = (Q, Γ, Σ, δ, q0, H)

Q    = 知识快照集合
Γ    = 所有实体+关系
Σ    = 新事实
δ    : Q × Σ → Q (推导引擎)
q0   = 初始状态
H    = 停机条件(连续N轮无新推论)
```

## 核心能力

| 能力 | 实现 | 说明 |
|------|------|------|
| 快照 | snapshot() | 记录当前全部文件状态 |
| 差异 | diff() | 对比两次快照的变化 |
| 回放 | replay() | 列出所有快照时间线 |
